# VinuSwap

Pool-based token swapping contract, based on Uniswap v3. Notable differences:
- Custom discounts based on token balances
- Controller contract for fee collection and splitting
- Support for locking a position until a specified timestamp

# Installing

```
git clone https://github.com/Vita-Inu/VinuSwap-VinuChain
npm install
```

# Running Tests

```
npm run test
```
